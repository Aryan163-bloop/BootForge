Method 1 • Recommended

Install Bootstrap using CDN

CDN (Content Delivery Network) is the fastest and easiest way to start using Bootstrap. Simply copy the code below and paste it into your project.

index.html 
<head>

<link
href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css"
rel="stylesheet">

</head>

<body>


<script
src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js">
</script>

</body>

CDN Overview

  • ✅ Fastest setup
  • ✅ No downloads
  • ✅ Perfect for beginners
  • ✅ Always latest version
  • ❌ Internet required
Recommendation
Use the CDN method while learning Bootstrap or building small projects.

Method 2 • Offline Development

Install Bootstrap by Downloading Files

Download Bootstrap directly to your computer. This method is useful when you need offline development or complete control over framework files.

Download Overview

  • ✅ Internet only once
  • ✅ Works completely offline
  • ✅ Full control over files
  • ❌ Manual updates required
  • ⭐ Best for medium projects

Install Bootstrap with npm

Build Your First Bootstrap Page

Let's put everything together by creating a simple Bootstrap webpage using the CDN method.

index.html 
<!DOCTYPE html>
<html>

<head>

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css"
rel="stylesheet">

</head>

<body>

<div class="container py-5">

<h1 class="text-primary">

Hello Bootstrap!

</h1>

<button class="btn btn-success">

Click Me

</button>

</div>

</body>

</html>
